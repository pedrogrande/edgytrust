# Verification Rubric

This document defines the scoring criteria for the 6-dimension ontology.

## Dimensions (Total: 100 Points)

1. **Capability (25 pts)**
   - Technical Correctness (15)
   - Capability Match (10)

2. **Accountability (10 pts)**
   - Ownership Signals (5)
   - Handoff Quality (5)

3. **Quality (30 pts)**
   - Functional Correctness (15)
   - Sanctuary Culture (10)
   - Code Quality (5)

4. **Temporality (10 pts)**
   - Dependency Respect (5)
   - Event Logging (5)

5. **Context (10 pts)**
   - Context Efficiency (5)
   - Pattern Reuse (5)

6. **Artifact (15 pts)**
   - Completeness (10)
   - Traceability (5)

## Scoring Logic
- **Pass**: >= 80 points
- **Excellent**: >= 90 points
- **Consensus Trigger**: > 10 point divergence
